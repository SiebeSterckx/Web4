# Web 4 project

## Must install:
    Backend: npm install
    Frontend: npm install react-router-dom @types/react-router-dom


## Commands to start:
backend:

    prisma:

        prisma format
        npx prisma migrate dev --name init
        npx ts-node init-db.ts

    npm start


frontend:

    npm run dev
